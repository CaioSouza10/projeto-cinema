PROJETO CINEMA.

O que ele faz: Simula um site de busca por filmes, utilizando API da IMDB.

Tecnologias usadas: Python, Flask, Jinja2, CSS Grid/Flexbox.

Como rodar: Instalar o requests, o flask e dar o python app.py.


## ✨ Nova funcionalidade (v2)

- Integração com Google Gemini API
- Geração automática de insights sobre filmes
- Chat contextual por filme
- Uso de variáveis de ambiente para segurança
